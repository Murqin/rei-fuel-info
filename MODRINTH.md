# REI Fuel Info

Adds the missing **Fuel** category to REI for Minecraft 1.21.2+.

> **Note:** REI's built-in fuel category was disabled starting from MC 1.21.2. This mod restores that functionality.

## Usage

1. Open REI (`R` or `U` key)
2. Click on any fuel item (coal, wood, etc.)
3. Look for the **Fuel** category

## Requirements

- Minecraft **1.21.2+**
- Fabric Loader
- Fabric API
- REI (Roughly Enough Items)
